### 二、解决方案：PTvo 插件（500字）

**介绍 PTvo：**

PTvo 是我开发的一个 VS Code 插件，它的核心理念是：

> **在 VS Code 里直接预览和翻译文档，不需要离开编辑器**

**核心功能：**

1. **一键预览**
   - 点击编辑区右上角的图标
   - 在右侧面板打开预览视图
   - 自动跟随主题颜色

2. **AI 翻译**
   - 支持 DeepSeek、智谱 AI、通义千问等国产 AI
   - 价格便宜（DeepSeek 只要 ¥1/百万 token）
   - 速度快，翻译准确

3. **实时同步**
   - 编辑文件时预览自动更新
   - 切换文件时自动切换预览内容

### 五、如何安装使用（300字）

**步骤 1：安装插件**
```bash
# 从 GitHub 下载 .vsix 文件
code --install-extension ptvo-x.x.x.vsix
```

**步骤 2：获取 API Key**
- DeepSeek：https://platform.deepseek.com（推荐）
- 智谱 AI：https://open.bigmodel.cn
- 新用户都有免费额度

**步骤 3：配置插件**
1. 打开 VS Code 设置
2. 搜索 "PTvo"
3. 填入 API Key

**步骤 4：开始使用**
- 打开任意文件
- 点击右上角的预览图标
- 点击 Translate 按钮
